Cartoon Network - logo
Conversion from Flash to HTML5
The games were originally developed by Cartoon Network in Flash.With a little help from Fundemic developers, the residents of Elmore made a leap into HTML5. Ensuring smooth performance in all browsers was key to successful porting. Along rebuilding code in JS, our developers used some clever tricks to keep fluid, frame-by-frame animations that made original games so unique.
